# 航空无人机货运产品线规格表

本文档定义了当前运营的所有货运无人机型号及其技术参数。

## 1. Swift-10 (轻型敏捷型)
- **适用场景**：同城急送、医疗样本、生鲜保鲜。
- **最大载重**：10 kg
- **货舱尺寸**：40cm x 30cm x 20cm
- **最大航程**：50 km
- **巡航速度**：60 km/h
- **起降要求**：平整硬化地面，需 2m x 2m 空间。
- **计费标准**：首重(1kg) 25元 + 续重 2元/kg。

## 2. Cargo-Max (城市物流主力)
- **适用场景**：电商包裹、电子产品、当日达快递。
- **最大载重**：50 kg
- **货舱尺寸**：80cm x 60cm x 50cm
- **最大航程**：150 km
- **巡航速度**：80 km/h
- **起降要求**：小型停机坪或空旷广场，需 5m x 5m 空间。
- **计费标准**：按距离阶梯计费，100km内 150元/票。

## 3. Titan-200 (重型工业型)
- **适用场景**：应急救援物资、工业零部件、偏远地区补给。
- **最大载重**：200 kg
- **货舱尺寸**：120cm x 100cm x 80cm
- **最大航程**：300 km
- **巡航速度**：120 km/h
- **起降要求**：专用起降场（LZ），需 10m x 10m 空间，地面承重需大于 500kg。
- **计费标准**：需人工询价，受天气和空域影响较大。

## 注意事项
- 所有机型均不支持搭载人员。
- 载重若包含包装箱重量，请预留 5% 的余量。
