# 危险品与禁运品管理规范 (依据 CAAC 及 IATA 标准)

## 绝对禁运品 (严禁托运)
以下物品**绝对禁止**通过任何无人机货运渠道运输：
1.  **爆炸物**：烟花爆竹、弹药、雷管。
2.  **气体**：压缩气瓶（如氧气瓶）、丁烷气罐、喷雾杀虫剂。
3.  **易燃液体**：汽油、柴油、酒精（含量 > 70%）、油漆稀料。
4.  **剧毒物品**：氰化物、砷、农药原液。
5.  **强腐蚀性**：硫酸、盐酸、蓄电池（含游离电解液）。

## 限制性运输品 (需审批)
以下物品需提前 24 小时报备并获得空管许可：
1.  **锂电池**：额定能量超过 100Wh 但小于 160Wh 的设备（如大型无人机电池、专业摄影器材）。必须做好绝缘处理。
2.  **干冰**：用于冷藏，净重不得超过 2.5kg，且包装必须有透气孔。
3.  **磁性物质**：扬声器、大型电机，需进行磁屏蔽包装，防止干扰无人机罗盘。

## 活体动物运输
- **活禽/牲畜**：禁止。
- **宠物（猫狗）**：仅限 Cargo-Max 机型，需配备专用透气笼具，且气温需在 0°C 至 30°C 之间方可飞行。

## 拒收后果
若隐瞒托运禁运品，公司将处以 5000 元罚款，并移交公安机关处理。AI 客服在遇到相关询问时，必须明确拒绝并提示法律风险。
