# 异常事件处理与理赔 SOP

## 1. 航班延误定义
- **轻度延误**：到达时间晚于预计时间 30 分钟以内。通常由于临时空中交通管制（ATC）导致。
- **重度延误**：晚于预计时间 2 小时以上。通常由恶劣天气（暴雨、大风 > 12m/s）或无人机故障导致。

## 2. 货物破损/丢失理赔
若用户反馈货物破损或丢失，请按以下步骤引导：

### 证据收集
- 要求用户提供：运单号、货物开箱视频、破损部位高清照片。
- 核实外包装是否完好（若外包装完好内物损，可能是发货方包装不当）。

### 赔偿标准
| 货物类型 | 保价情况 | 赔偿方案 |
| :--- | :--- | :--- |
| **普通货物** | 未保价 | 退还运费，并赔偿运费的 3 倍（上限 500 元）。 |
| **普通货物** | 已保价 | 按实际损失金额赔偿，最高不超过保价金额。 |
| **贵重品** | 未声明 | 最高赔偿 1000 元（建议用户今后对贵重物品进行保价）。 |

## 3. 备降与返航
- **天气原因备降**：如遇突发雷暴，无人机将自动备降至最近的中转站。系统会自动推送短信通知收件人。
- **收货方拒收**：若收件人拒绝收货，货物将自动返航至始发站，返航运费由发件方承担。

## 4. 客服话术指引
- 当用户情绪激动时，请回复：“非常抱歉给您带来不便，我们已启动紧急调查程序，工单号 {{ticket_id}} 将在 30 分钟内由专员跟进。”
